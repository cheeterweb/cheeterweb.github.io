Was ist eine KI?

Eine KI (Künstliche Intelligenz) ist damit sich ein Roboter bewegen kann, Ein Zombie in Videospielen weiß wo du bist, Dein Charakter auf WASD reagiert, Du tasten einstellen kannst, Dein Server bestimmte Aktionen ausführen kann und so weiter...
